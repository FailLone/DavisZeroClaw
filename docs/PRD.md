# DavisZeroClaw 产品需求文档 (PRD)

## 1. 产品概述

### 1.1 产品代号与定位
- **产品代号**：DavisZeroClaw
- **产品定位**：一个基于 **ZeroClaws** 底层架构构建的、面向中国大陆用户的轻量级本地 AI 智能体中枢。它运行在用户的个人电脑（如 Mac/迷你主机）上，通过 Siri/HomePod 和快捷指令提供语音入口，深度打通 Home Assistant 等智能家居生态，并具备基于本地记忆的自我进化能力。

### 1.2 背景与核心价值
传统的智能家居缺乏真正的“理解力”，而云端大模型又缺乏对本地家庭状态的感知与控制权。
本产品借助 ZeroClaws 极低资源占用的“Agent OS”特性，在本地构建一个“家庭大脑”：
1. **轻量本地化**：利用 ZeroClaws Daemon 常驻后台，内存占用极低（<5MB），无需庞大复杂的环境配置。
2. **无缝苹果生态**：利用 ZeroClaws Gateway 提供本地 Webhook，完美对接 iOS Shortcuts，实现 Siri/HomePod 随声唤起。
3. **状态感知与推断**：通过 ZeroClaws Memory 系统托管家庭状态，从被动响应升级为主动管家。

## 2. 核心功能模块

### 2.1 桌面宿主与引擎层 (基于 ZeroClaws Daemon)
- **静默常驻**：作为桌面后台应用或菜单栏工具运行。
- **任务调度**：利用 Cron/Heartbeat 机制，定期拉取家庭状态或执行自动化脚本。
- **模型接入**：通过 ZeroClaws Provider 接口，无缝接入支持 OpenAI 格式的中国大陆大模型（如 DeepSeek、通义千问）或本地模型（Ollama）。

### 2.2 远程控制与通信入口 (基于 ZeroClaws Gateway)
- **本地 Webhook 服务**：默认开启本地端口，专门接收来自内网的快捷指令请求。
- **快捷指令模板**：提供官方 iOS Shortcuts 模板，支持“语音转文本发送”、“结果朗读”、“App 辅助跳转（如外卖）”。
- **云中继 (规划中)**：为解决外网访问，规划轻量级内网穿透或云端消息转发机制。

### 2.3 智能家居连接器 (基于 ZeroClaws Tools 与 MCP)
- **Home Assistant 集成 (MCP 优先)**：优先通过 **Home Assistant 官方 MCP Server** 协议对接，实现零配置、高可靠的设备控制与状态感知。作为备份，保留封装 `ha_tool` 的能力。
- **自然语言控制**：大模型负责将模糊指令（“我有点冷”）转化为精确的 Tool 调用。借助 MCP 协议，Agent 能自动获得 HA 导出的能力描述（Prompts/Tools）。

### 2.4 生活服务辅助 (基于 ZeroClaws Tools)
- **快递查询**：提供查询工具，整合国内常用快递接口。
- **外卖等跨应用服务辅助 (App Intents 优先)**：理解点餐等复杂意图，通过返回结构化参数让 iOS 快捷指令调用目标 App 的 **App Intents** (如美团“再来一单”)。若无 Intent 支持，降级返回 URL Scheme 拉起对应 App，形成半自动或全自动闭环。

### 2.5 自我进化与记忆 (基于 ZeroClaws Memory)
- **家庭状态图谱**：定时获取并持久化当前传感器、设备状态到本地 SQLite。
- **习惯学习**：记录用户的高频指令与时间点（如每晚 10 点关灯）。
- **主动建议**：当 Daemon 推断出某种场景模式时，主动推送通知给用户询问是否创建新的自动化规则。

## 3. MVP (V1.0) 交付范围

1. **核心引擎**：打包集成了 ZeroClaws 核心的 macOS 菜单栏应用。
2. **模型接入**：支持配置 1-2 家国内大模型 API。
3. **Siri 链路**：提供标准的 iOS 快捷指令配置教程与模板，利用 App Intents 实现应用深度控制。
4. **家居控制**：实现与局域网内 Home Assistant 的对接，支持基础的灯光、开关控制。
5. **对话与查询**：支持自然语言问答和快递查询工具。

## 4. 目标用户
- 拥有 Mac 或微型主机作为常开设备的极客/开发者。
- 苹果生态重度用户（iPhone, HomePod, Apple Watch）。
- 已部署 Home Assistant，希望获得更智能、模糊语义控制能力的智能家居玩家。

## 5. 长期规划与非目标 (V2.0+)
- **不开发独立 iOS App (非目标)**：为保持系统的轻量化，坚决不开发维护独立的 iOS App，核心交互全部交给系统的 Siri 和 Shortcuts 完成。
- **桌面级视觉自动化引擎 (V2 核心特性)**：引入 `ComputerUseTool`，结合多模态大模型 (VLM) 和 macOS 的“iPhone 镜像”功能，实现对无 API 桌面软件或顽固手机 App 的终极屏幕级物理操作，达到“所见即所得”的全自动接管。
